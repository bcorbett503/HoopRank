# HoopRank — Frontend (Vite + React + TypeScript)

**WNBA-only mock data**, Profile onboarding, and 1–5 HoopRank scale.

## Quickstart

```bash
cd hooprank-frontend
npm install
npm run dev
# http://localhost:5173
```

## Notes
- First login redirects to **/profile/setup** (age, ZIP, height, position).
- Rankings now display **ZIP, Height, HoopRank (1–5)**.
- Mock data only includes WNBA players per request.
- Rating deltas tuned for **small 1–5 changes**.
